MPages Help Pages : Adding Information to ICU Summary (52330603)
This page last changed on Aug 14, 2011 by JP3026.


You can post directly to PowerChartÂ® from the ICU Summary MPage. You have several options for entering information into the patient chart listed below. Follow one or more of the options to enter the appropriate data:

Option One:

  1. Navigate to the MPage Summary view.
  2. Click the title of a selected component to launch the appropriate PowerChart tab. The PowerChart tab is used to enter data generally.

Option Two:

  1. Navigate to the MPage Summary view.
  2. If a "+Add" is displayed to the right of the component title, you can select it to launch the PowerChart® module used to enter data for that component. Some examples are listed below:
    1. Vital Signs component - "+Add" launches the Ad-Hoc Charting window where the PowerForms used for charting can be selected.
    2. Allergies component - "+Add" launches the Add Allergies module for entering new allergies.
    3. Documents component - "+Add" launches the PowerNote Documentation tab for selecting the specific encounter pathway.
Note
Not all components have this capability. For those that do, the capability must be set up and defined in the Bedrock® Wizard.

Option Three:

  1. Navigate to the MPage Summary view.
  2. If multiple items have been defined in the Bedrock Wizard, a drop down arrow is displayed to the right of the "+Add" option for a specific component.
  3. To display the list of specific forms, position your mouse over the drop down icon (or click it). From there, you can directly launch the notes or orders.
  4. Click the appropriate item to open the window for entering data.


When you click "+Add", the system can also launch one of the PowerChartcomponents listed below for entering and modifying data:

  • Medication Reconciliation
  • Patient Education
  • Follow Up
Note
If the Win32 Patient Education module is not implemented, you can configure the Bedrock Wizard to pull in documented data from PowerForms.
