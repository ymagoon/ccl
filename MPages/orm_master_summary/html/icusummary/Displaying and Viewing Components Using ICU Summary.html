R2 MPages Help Pages : Displaying and Viewing Components Using ICU Summary
This page last changed on Aug 18, 2010 by JP3026.


There are a couple of ways for displaying and viewing information on the ICU Summary MPage. You can follow one or more of the options to display the appropriate data.

Option One:

  1. Navigate to the MPage Summary view.
  2. If a "Add" is displayed to the right of the component title, select it to expand the component. Below are some examples describing the expanded component data.
    1. Lines, Tubes, and Drains component - "Add" expands the component window where the procedure and the documented date and time are displayed.
    2. Allergies component - "Add" expands the Allergies component, which lists each known drug and food allergy for a patient.
    3. Labs component - "Add" expands the Labs component where the result categories like CO2, BUN, and WBC are displayed. The latest and previous results are displayed under a column for the result category as well. A graph can be displayed for a result category when it has an underlined link. For example, if you click CO2 (it has a blue underlined link), the system displays a graph with the MEq/L (milliequivalents/liter) on the Y axis and the date and time on the X axis.
Note
Not all components have this capability. For those that do, the capability must be set up and defined in the Bedrock® Wizard.


Option Two:

  1. Navigate to the MPage Summary view.
  2. If it is not already expanded (you see a minus sign next to the component title), expand the component as described in Option One above.
  3. Position your mouse over a result row or column to display a pop-up window containing additional details. See the MPage components described below.
    Note
    These are just a few of those offered.
    1. Lines, Tubes, and Drains component - The units of output for a line, tube, or drain are displayed, and the documented date and time are displayed as well.
    2. Allergies component - The Allergy Name, Reaction, Severity, Status, Onset Date, and PowerChartÂ® comments are displayed.
    3. Labs component - The Normal Low, Normal High, Critical Low, and Critical High values are displayed for a category column like the Latest or Previous result.
