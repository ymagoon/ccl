MPages Help Pages : Overview of ICU Summary (52330601)
This page last changed on Aug 14, 2011 by JP3026.

MPages provides the ability to create a customized view of Cerner Millennium® data in an HTML format. This MPage is focused on providing a graph view of key clinical information related to patients in all types of intensive critical care units, including NICU, PICU, etc. Additions to this version include actionability, framework enhancements, and standard components as of March, 2010. The graphical clinical view can use a number of the standard MPage design components for Problems, Diagnoses, Allergies, Vital Signs, Measurements and Weights, Medications, Intake and Output, Microbiology, Diagnostics, Labs, Lines, Tubes, and Drains, Documents, Patient Information, CDC Growth Chart, Procedure History, Notes and Reminders, Immunizations, New Order Entry, Outstanding Orders, Restraints, Pathology, and the ICU Graph. The Cerner Millennium® Bedrock Wizard allows you to define the components that display on the page, the order of the way components display, and the the columns on the graph. From the summary page, clinicians can add documentation, add orders, link directly to "add" modules within the chart for some components, and customize their view.
