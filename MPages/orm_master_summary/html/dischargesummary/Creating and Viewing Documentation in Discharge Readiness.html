MPages Help Pages : Creating and Viewing Documentation in Discharge Readiness
This page last changed on Dec 14, 2010 by JP3026.

Once you have displayed a patient and clicked the Discharge Readiness MPages tab in PowerChartÂ®, the Discharge Summary MPages view is displayed. To add documentation, complete the following steps:

  1. Search for the desired encounter pathway you want to document against. To complete this step, click "+Add" on the Documents component to launch the documentation tab in PowerChart.
  2. To launch a specific encounter pathway, select or position your mouse over the down arrow icon. This action displays the Favorites Encounter Pathways list.
  3. Click the pathway name you want.

To view documentation, complete the following steps:

  1. After expanding the Documents component, select the documented Date/Time link next to the document name you want to view. The system displays the clinical note you selected.
