Release Considerations
Beginning with MPages Content Package 51872 ( MPages Component Library 2.4 (May 2011)) and MPages Bedrock Configuration Content Packages 51894, 51895, 51896, and 51897 (MPages: XXXXX Summary Configuration 2.0 (May 2011)) and its replacement, the functionality for the modify indicator for the Labs component described below became available. Beginning with MPages Content Package XXXXX ( MPages Component Library 4.0 (July 2011)) and MPages Bedrock Configuration Content Package XXXXX, XXXXX, XXXXX, and XXXXX (MPages: XXXXX Summary Configuration 4.0 (July 2011)) and its replacement, the functionality described below became available for most MPages components. This enhancement affects the components for Labs, Discharge Activities, Discharge Plan, Measurements and Weights, Patient Assessment, Psychosocial Factors, Discharge Results, Social Discharge, Vital Signs, Diagnostics, Documents, Microbiology, Pathology, Flagged Events, CDC Growth Chart, Immunizations, Lines Tubes and Drains, Patient Background, Patient Information, Respiratory Assessments, Respiratory Treatments, Oxygenation and Ventilation, Ventilation Monitoring, and Intake and Output.

When a result has been modified, a special modify indicator (blue triangle) is displayed to the right of the most recent result on the  MPages view. This indicator can be the result of one or more of the following items:

  • The result value was corrected.
  • The date or time of the result was changed.
  • A comment was added to a result after it was signed.

To view the details of the modification, complete the following steps.

  1. Hover over the result to display details including the status of "Modified".
  2. Click on the result. The Result Details window is displayed with the details of the result and historical information related to the modification.
Note
When you hover over a modified result, the hover displays a window that should contain a row indicating the status is "Modified". Also, you can access the Result Details window for a clinical result by clicking any result on the page.
