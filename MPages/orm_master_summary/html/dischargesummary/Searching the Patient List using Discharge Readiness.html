MPages Help Pages : Searching the Patient List using Discharge Readiness
This page last changed on Aug 09, 2011 by JP3026.

Once you are logged-on, a patient list for the Discharge Readiness MPages view is displayed. The default patient list for a clinician is initially displayed. The patient list can be changed to display your preferences defined in PowerChart. See the topic Displaying the Patient List using Discharge Readiness for more information. To search for a patient or a group of patients from the currently displayed patient list, complete the following steps:

  1. Click the Search tab. Within the patient search field, enter the patient's "last name", the "last name, first name", the "first name<space>last name" or the patient's medical record number.
    Note
    The more characters you enter in the Search box, the more the system narrows your search results. For example, if you enter the last name characters of "JOH" only, the system returns patients with the last name of JOHANESS, JOHNS, JOHNSON, etc. However, if you enter "JOHNSON" in the Search box, the system returns the patients with at least a last name of JOHNSON. Also, you can enter the first name separated by a space followed by the last name. For example, if you entered "RANDY JOHNSON", the system returns a list of the patients with the first and last name of RANDY JOHNSON. 
  2. Click Search. The system displays a patient list based on the search criteria you specified.
