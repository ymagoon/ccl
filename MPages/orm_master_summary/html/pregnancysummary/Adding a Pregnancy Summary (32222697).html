MPages Help Pages : Adding a Pregnancy Summary (32222697)
This page last changed on Aug 16, 2011 by JP3026.

After launching the Pregnancy Summary from PowerChart(Powerchart.exe), the system displays the following message: There are currently no active pregnancies for this patient. Complete the following steps to Add a Pregnancy Summary to your patient:

  1. Click the Add Pregnancy button. The Add Pregnancy window opens.
  2. Enter the Responsible Provider, Onset: Date, and Confirmation information.
  3. Enter any comments you would like recorded.
  4. Click OK.

Once you have added the pregnancy information, the system displays the MPages view.

Note:
The page timeframe is determined by the pregnancy onset date and time.
