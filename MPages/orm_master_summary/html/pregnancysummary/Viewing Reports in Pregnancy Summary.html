MPages Help Pages : Viewing Reports in Pregnancy Summary
This page last changed on Apr 12, 2010 by MB9512.

Microbiology
From the Microbiology component in Pregnancy Summary click the collection date and time of the report you want to view. The system opens the report in Micro Viewer or displays the report in a Rich Text Format (RTF) view. Close the report when you have finished viewing it.

Pathology
From the Pathology component in Pregnancy Summary click the verified date and time of the report you want to view. The system opens the report. Close the report when you have finished viewing it.

Diagnostics
From the Diagnostics component in Pregnancy Summary click the date and time of the diagnostics report you want to view. The system opens the report. Close the report when you have finished viewing it.
