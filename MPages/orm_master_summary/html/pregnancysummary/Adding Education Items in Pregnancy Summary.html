MPages Help Pages : Adding Education Items in Pregnancy Summary
This page last changed on Aug 15, 2011 by JP3026.

To add education items from the Education and Counseling component of the Pregnancy Summary MPages view, complete the following steps:

  1. Click the Add button (+Add) next to the Education and Counseling heading. The system opens the defined PowerForms view.
    Note
    You can define the PowerForms view you want in Cerner Bedrock. For more information see the All About Pregnancy Summary MPages Reference Page.
  2. Enter the appropriate education information.
  3. Save and close the PowerForms view.
