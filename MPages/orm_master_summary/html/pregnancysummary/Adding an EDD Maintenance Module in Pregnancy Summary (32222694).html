MPages Help Pages : Adding an EDD Maintenance Module in Pregnancy Summary (32222694)
This page last changed on Aug 16, 2011 by JP3026.

To add an Estimated Due Date (EDD) Maintenance Module complete the following steps:

  1. In the EDD Maintenance component header, click Add. The EDD Maintenance Module window opens.
  2. The system displays the EDD date as a link. Select the date you want.
  3. EDD Maintenance Tool opens in modify mode.
  4. Select the method by which the EDD was calculated (last menstrual period, ultrasound, or advanced reproductive technology) and the related details.  The associated details vary based on the EDD calculation method selected. 
  5. Select whether the EDD is initial, final or authoritative.
    Note:
      The system uses the status when making EDD calculations.  The EDD can be updated/reassessed due to subsequent ultrasounds.
  6. Click OK.
