Release Considerations
Beginning with MPages Content Package 55241 (MPages (US): Quick Orders and Charges (QOC) MPages View (November 2011)) and its replacement, the quick orders functionality described in this Reference Page became available. This particular package is available to testing partners only. Please contact your CRE if you want to become a development partner.

The Quick Orders and Charges MPages view is focused on providing hospital clinicians with the most commonly placed orders using public folders. Also, the view allows clinicians to select predefined clinical events for medications, laboratory, CT scans, X-Rays, etc., that can be placed on the New Order Entry scratch pad for signature. Multiple views of order folders and standard components can be built and added to a single Quick Orders and Charges MPages view. In addition to signing orders, this view supports selecting orders, associating orders to a documented diagnoses, viewing outstanding (incomplete) orders, and managing a patient's problem list through the Consolidated Problems component.
