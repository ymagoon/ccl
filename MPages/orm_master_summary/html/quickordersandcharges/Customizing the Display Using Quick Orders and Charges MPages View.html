Release Considerations
Beginning with MPages Content Package 55241 (MPages (US): Quick Orders and Charges (QOC) MPages View (November 2011)) and its replacement, the quick orders functionality described in this Reference Page became available. This particular package is available to testing partners only. Please contact your CRE if you want to become a development partner.

You can rearrange the order the MPages components are displayed, and you can define the default behavior for each one. Complete the steps below to customize the component display:

  1. Click the Customize link in the upper right-hand corner of the MPages view. The system displays the User Customization page where you can adjust the component behavior.
  2. Left-click a component and drag it to its new location or column.
    Note
    You can change the MPages layout to contain up to three columns. You change the layout from a three column view to a two column view by dragging and dropping the components into the appropriate number of columns. Components with an up arrow icon are displayed as being collapsed, and components with a down arrow icon are displayed as being expanded. To change the default behavior for a component, click the appropriate expand or collapse icons. To clear the previously defined preferences and return to the default view defined in the Bedrock Wizard, click Clear Preferences. To expand or collapse every component when the MPages view is displayed, click Expand All or Collapse All. When you position the mouse over the expand icon, the system displays "Expand". When you position the mouse over the collapse icon, the system displays "Collapse".
  3. When you have completed your modifications, click Save Preferences.
