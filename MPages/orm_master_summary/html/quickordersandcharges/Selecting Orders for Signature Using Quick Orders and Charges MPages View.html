MPages Help Pages : Selecting Orders for Signature Using Quick Orders and Charges MPages View
This page last changed on Nov 10, 2011 by JP3026.

Release Considerations
Beginning with MPages Content Package 55241 (MPages (US): Quick Orders and Charges (QOC) MPages View (November 2011)) and its replacement, the quick orders functionality described in this Reference Page became available. This particular package is available to testing partners only. Please contact your CRE if you want to become a development partner.

From this MPages view, you can select the orderables you want to sign on the New Order Entry component scratchpad. To select the orderables from the predefined folders, complete the following steps:

  1. Navigate to the MPages view in PowerChart.
  2. From the View Selector list (drop down) at the top of the page, select the predefined view you want.
    Note
    The selectable views are defined on the Quick Orders Configuration page, which is accessible through the Organizer view in PowerChart.
  3. If an orderable category like "Quick Medications" is collapsed, click the down arrow to the right of the title to expand the list.
  4. Select the orderables from the expanded list to send the orderables you want for signature to the scratchpad in the New Order Entry component.
    Note
    See the instructions beginning with Step # 5b3 below for information on how to sign the order.
  5. If a "+" plus sign is displayed to the right of a component title, select it to expand the component. Below are some examples describing the expanded component data.
    1. Consolidated Problems - If a search box is displayed below the component header, enter a few characters of a nomenclature search term to display a matching list. Select the result you want from the drop down list, and the component data is displayed below it.
    2. New Order Entry component - You can sign results and review unsigned orders for tabs like Medications, Laboratory, Imaging, Billing, and other user defined areas. The following example describes how this component is used.
      1. Select a tab in the component like Labs. A list of the Laboratory categories is displayed below the component tab.
      2. Click a category row like CDC. The system displays the outstanding orders below the categories in a list.
      3. If you want to sign an order, select those you want from the list below the categories. For example, you could select multiple orders by clicking the box for Chest CT W/O Contrast, Chest XR 2 Views, Chest XR 1 View if they are displayed in the Orders for Signature list.
        Note
        You can click multiple items from every tab first, and then the Orders for Signature list would contain every order you selected from those tabs. This step would allow you to sign multiple orders.
      4. Click the Sign button. The system returns you to the MPages view.
        Note
        You must have the appropriate privileges to sign an order. If you would prefer to review the unsigned orders before you sign those in the list, click the DX Table button. The system displays the Diagnosis Table for Unsigned Orders window. After you review the unsigned orders, click the Home button at the top of the window. You are returned to the MPages view. You can then sign those you want by selecting them from the list. Orders are removed from the scratchpad once they are signed. The Outstanding Orders component refreshes itself as orders are signed, and it displays the appropriate orders (as defined through Bedrock).
