Release Considerations
Beginning with MPages Content Package 55241 (MPages (US): Quick Orders and Charges (QOC) MPages View (November 2011)) and its replacement, the quick orders functionality described in this Reference Page became available. This particular package is available to testing partners only. Please contact your CRE if you want to become a development partner.

If you attempt to close the page or navigate to another chart without signing a pending order added to the New Order Entry scratchpad, the system displays a message indicating there are outstanding orders.

If you attempt to add a problem for which you do not have privileges, a message is displayed stating, "You do not have privileges to add the selected problem", and the problem is not added.
