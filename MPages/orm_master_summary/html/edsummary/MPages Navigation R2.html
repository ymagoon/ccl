Every individual component has an expand/collapse icon in the upper right corner. Components can default to being expanded or collapsed. This default behavior can be customized at the user level.

Moving up the navigation bar at the top, the first option is a search on the MPages view. You can open the search by clicking ctrl+F on your keyboard, or clicking the binocular icon.

You can use the the back, forward, and home icons to navigate to another screen within the MPages view, for example, if you view the latest office/clinic note for a patient, you can click the home icon to return to the main summary screen. You can use the back and forward arrows to return to pages you have already visited.

In the upper right hand corner of the screen you can click collapse all to collapse everything on the summary page, and then click expand all to expand everything open again.
