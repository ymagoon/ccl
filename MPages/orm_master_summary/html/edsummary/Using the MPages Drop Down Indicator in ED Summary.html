MPages Help Pages : Using the MPages Drop Down Indicator in ED Summary
This page last changed on Aug 19, 2011 by AF019864.

Release Considerations
Beginning with MPages Content Package 53866 ( MPages:MPages Component Library 4.0 (August 2011)) and MPages Bedrock Configuration Content Package 53867, 53868, 53869, 53870, 53871, 53872, 53873, 53874, and 53875 and its replacement, the functionality described below for the Drop Down Indicator became available for most MPages components.

When a result can be modified in an MPages view, a special drop down indicator (typically an upside down blue triangle) is displayed in the component header. The purpose of the icon is to provide a short list of PowerForms for easy access to charting against a new instance (in the case of the Vital Signs component), or to provide a short list of pre-completed PowerNotes you can save as a favorites template (in the case of the Documents component). This indicator means you can display the available categories to modify a patient's chart using a PowerForms interface, or you can display the details for a result category. For the drop down indicator to be displayed in the Documents component, you must select "Yes" for the Bedrock configuration question about enabling the Quick Add function. For the indicator to be displayed in the Vital Signs component, you must select the option for "PowerForms to Chart Vitals" in the Bedrock filter configuration.

To enter information or to see the details available using the Drop Down Indicator, complete the following steps:

Note
The drop down indicator does not require the same input for every MPages component. Some of them offer slightly different PowerForms. While MPages components are moving towards standardization, consider the steps below as basic guidance. For the Documents component to be displayed, you must create a pre-completed favorites note as opposed to an encounter pathways favorite. When you click the indicator, you are taken to the PowerNote tab of the chart with the selected template in context. Once you have signed the note, you are taken back to the Summary MPages view. If you refresh the page, the new information is displayed.
  1. Click the blue upside down triangle in the component header. The system displays a pop-up box with selectable values. For example, the pop-up box in the Vital Signs component contains elements for Oxygen Check, Basic Admission Information Adult, Post Fall Assessment, Post Fall Assessment EBN, Post Fall Assessment Pediatric, Advanced Directives, Ambulatory Vitals Height and Weight, etc.
  2. If you select Ambulatory Vitals Height and Weight from the list, the system displays a window allowing you to enter vital sign information like temperature, height, weight, etc. Enter the values you want, and click the diskette icon on the toolbar to save the information to the patientâ€™s chart. The system closes the window and you are returned to the MPages view.
    Note
    If you enter a value that is outside the Bedrock mapping or defined range, the system displays an error giving you the feasible and allowable range. Also, certain component toolbars offer a clickable icon to display result details, and some windows have clickable categories that flex the input values.
  3. To see only results, click the All Visits category name inside the Vital Signs component display on the MPages view. For example, you would click the blue underlined category name for Respiratory Rate or Height/Length. The system displays a graph with the results listed by date and time.
    Note
    You might need to hover over the category name before the blue line is displayed indicating there is a link to additional details. Also, certain component toolbars offer a clickable icon to display result details after you make a selection in Step # 2 above.
    Release Considerations
    Beginning with MPages Content Package 51872 ( MPages Component Library 2.4 (May 2011)) and MPages Bedrock Configuration Content Package 51895 (MPages: Inpatient Summary Configuration 2.0 (May 2011)) and its replacement, an abnormal indicator is displayed in the color scheme listed below for non-numeric or abnormal results.
    • Critical = bold red; red exclamation point icon
    • Abnormal High = bold orange; orange up icon
    • Abnormal Low = bold blue; blue down icon
  4. Once the results information window is displayed, click the X in the upper right hand portion of the window to close it. Then, the system returns you to the main window of the MPages view.
