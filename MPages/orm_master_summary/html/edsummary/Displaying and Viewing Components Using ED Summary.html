MPages Help Pages : Displaying and Viewing Components Using ED Summary
This page last changed on May 31, 2011 by JP3026.


There are a couple of ways for displaying and viewing information on the ED Summary MPage. You can follow one or more of the options to display the appropriate data.

Option One:

  1. Navigate to the MPage Summary view.
  2. If a "+" plus sign is displayed to the right of the component title, select it to expand the component. Below are some examples describing the expanded component data.
    1. Allergies component - "+" plus sign expands the Allergies component, which lists each known drug and food allergy for a patient.
    2. Labs component - "+" plus sign expands the Labs component where the result categories like CO2, BUN, and WBC are displayed. The latest and previous results are displayed under a column for the result category as well. A graph can be displayed for a result category when it has an underlined link. For example, if you click CO2 (it has a blue underlined link), the system displays a graph with the MEq/L (milliequivalents/liter) on the Y axis and the date and time on the X axis.
      Release Considerations
      Beginning with MPages Content Package 51872 ( MPages Component Library 2.4 (May 2011)) and MPages Bedrock Configuration Content Package 51894 (MPages: ED Summary Configuration 2.0 (May 2011)) and its replacement, you can launch the Results viewer containing additional details by clicking a result column in the Labs component. Also, if a lab result has a status of "Modified", the system displays a modify icon (blue triangle) to the right of the most recent version of a result. The status of "Modified" displays when the result value was corrected, or when the date/time of the result was modified, or when a comment was added to a result after it was signed. Also, the Abnormal icon is displayed in the Labs, and Vital Signs components for non-numeric or abnormal results.
    3. Problems and Diagnosis component - If a search box is displayed below the component header, enter a few characters of a nomenclature search term to display a matching list. Select the result you want from the drop down list, and the component data is displayed below it.
      Release Considerations
      The Quick Add Search functionality described above was added with Content Package 50521 (ED Summary MPages View 2.3 (Feb 2011)). If you plan to use this functionality, make sure server 396 is running and the correct preferences are configured in Preference Maintenance Tool (PrefMaint.exe). For details on the functionality and the Preference Maintenance Tool configuration, see the Problems and Diagnosis design component pages.

Option Two:

  1. Navigate to the MPage Summary view.
  2. If it is not already expanded (you see a minus sign next to the component title), expand the component as described in Option One above.
  3. Position your mouse over a result row or column to display a pop-up window containing additional details. See the MPage components described below.
    Note
    These are just a few of those offered.
    1. Allergies component - The Allergy Name, Reaction, Severity, Status, Onset Date, and PowerChart® comments are displayed.
    2. Labs component - The Normal Low, Normal High, Critical Low, and Critical High values are displayed for a category column like the Latest or Previous result.
