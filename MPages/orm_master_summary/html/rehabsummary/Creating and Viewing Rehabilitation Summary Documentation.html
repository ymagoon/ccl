MPages Help Pages : Creating and Viewing Rehabilitation Summary Documentation
This page last changed on Jun 26, 2011 by JP3026.

Once you have displayed a patient and clicked the Rehabilitation Summary MPages tab in PowerChart, the Rehabilitation Summary MPages view is displayed. You can update a PowerNote and view documented results using this component. To add documentation, complete the following steps:

  1. Click a results row that contains a blue underlined link, which means there are documented results. The Document Viewer window is displayed.
  2. To update the PowerNote on the patient's chart, click the fourth icon over from the left on the toolbar in the Document Viewer window. The system launches the Open Note window.
  3. Select either Current Encounter or All Encounters on the Existing tab. Select the note you want from the list and click Make Correction. The system displays the Open Note editor where you can modify the note.
    Note
    The Open Note window contains tabs for Encounter Pathway, Existing, Precompleted, Catalog, Recent, and Favorites.
  4. Close the open windows to return to the MPages view.
