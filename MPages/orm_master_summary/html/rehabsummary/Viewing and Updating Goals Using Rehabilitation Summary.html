MPages Help Pages : Viewing and Updating Goals Using Rehabilitation Summary
This page last changed on Jun 27, 2011 by JP3026.

Once you have displayed a patient and clicked the Rehabilitation Summary MPages tab in PowerChart, the Rehabilitation Summary MPages view is displayed. To add or review the patient's goals, complete the following steps:

Using the Ad-Hoc Charting Feature

  1. To document results by PowerChart category, click "+Add" on the Current Status component. The system launches the Ad-Hoc Charting window in PowerChart.
  2. Double-Click a charting category. The system populates the right side of the Ad-Hoc Charting window with a selectable list of items that relate to the category. For example, if you double-clicked Respiratory Care as a charting category, the system would add selectable values for Aerosol Therapy, Chest Physiotherapy, Incentive Spironmetry, Oxygen Therapy, etc.
  3. Select an item from the right-side of the window and click Chart. The system displays the charting details window for the category you selected. If you selected Aersol Therapy, the Aersol Therapy details window would display. Then, you would select values that pertained to the treatment, For example, you might need to enter a selection for the Delivery Device, Treatment Route, FIO2, Peak Inspiratory Pressure, Participation Level in Treatment, Patient Effort, etc.
    Note
    If you click Close, the system displays a message indicating you have selected one or more items to chart. It asks whether you want to chart the selected item and provides a Yes or No button. If you select Yes, the system displays the Details window. If you select No, you are returned to the Ad-Hoc Charting window.
  4. Fill-In the requested details and click Save on the toolbar of the charting details window. The system adds the charting details into the patients electronic medical record in PowerChart, and you are returned to the MPages view.

Using the Drop Down Indicator Feature

  1. To indicate you have reviewed the patient's current goals, click the blue triangle icon in the component header. The system displays a drop down list (menu) with selectable categories like SLP Inpatient Daily Documentation or Interdisciplinary Team Conference Note.
  2. Select an item from the list. If you selected SLP Inpatient Daily Documentation, the PowerChart Review window is displayed. You can then update the documentation by selecting whether you have reviewed the short term goals. Also, you can select values for the session notes such as Pain Interventions, Pain Notifications, etc, and you can update the section for Treatment Provided This Date.
  3. If you indicated short term goals were reviewed in Step # 2 above, the system displays the short term goals input window. You can update a row for a certain goal number, by clicking each cell in a row and completing the requested details as you are prompted. When you have selected the appropriate values, and you have entered the appropriate information, click OK. You are returned to the short term goals input window.
  4. When you have completed the documentation, click Save on the toolbar. The system closes the Review window and returns you to the Rehabilitation Summary MPages view.

Displaying and Filtering Goal Component Categories

By making a selection on the appropriate component row that contains a filter list (drop-down), you can filter and display Nursing goals by Met, Not Met or Both for example.

  1. To filter Nursing goals by those that are Met, select Met from the list (drop-down) on the Nursing category row in the Goals component. The system displays those goals matching your selection below the category row.
  2. To display the details for a goal listed below the category, double-click the row.
    Note
    You can perform this function for any category row in the Goals component like OT, PT, SLP, etc. Expand the category row and double-click the goal you want to display the details.
