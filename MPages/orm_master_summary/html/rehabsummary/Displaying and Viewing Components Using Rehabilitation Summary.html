MPages Help Pages : Displaying and Viewing Components Using Rehabilitation Summary
This page last changed on Jun 26, 2011 by JP3026.

There are a couple of ways for displaying and viewing information on the Rehabilitation Summary MPages view. You can use one or more of the options described below to display the appropriate data.

Option One:

  1. Navigate to the MPage Summary view.
  2. If a "+" plus sign is displayed to the right of the component title, select it to expand the component. Below are some examples describing the expanded component data.
    1. Lines, Tubes, and Drains component - "+" plus sign expands the component window where the procedure and the documented date and time are displayed.
    2. Allergies component - "+" plus sign expands the Allergies component, which lists each known drug and food allergy for a patient.
    3. Labs component - "+" plus sign expands the Labs component where the result categories like CO2, BUN, and WBC are displayed. The latest and previous results are displayed under a column for the result category as well. A graph can be displayed for a result category when it has an underlined link. For example, if you click CO2 (it has a blue underlined link), the system displays a graph with the MEq/L (milliequivalents/liter) on the Y axis and the date and time on the X axis.
      Note
      Not all components have this capability. For those that do, the capability must be set up and defined in the Bedrock® Wizard.
      Release Considerations
      Beginning with MPages Content Package 51872 ( MPages Component Library 2.4 (May 2011)) and MPages Bedrock Configuration Content Package 51895 (MPages: Inpatient Summary Configuration 2.0 (May 2011)) and its replacement, you can launch the Results viewer containing additional details by clicking a result column in the Labs component. Also, if a lab result has a status of "Modified", the system displays a modify icon (blue triangle) to the right of the most recent version of a result. The status of "Modified" displays when the result value was corrected, or when the date/time of the result was modified, or when a comment was added to a result after it was signed. Also, the Abnormal icon is displayed in the Labs, and Vital Signs components for non-numeric or abnormal results. Beginning with MPages Content Package XXXXX ( MPages Component Library 4.0 (July 2011)) and MPages Bedrock Configuration Content Package XXXXX, XXXXX, XXXXX, and XXXXX (MPages: XXXXX Summary Configuration 4.0 (July 2011)) and its replacement, the Forwarding enhancement became available for the Labs and Vital Signs components.
      1. To display a result in the Result viewer, double-click a result row (for example BP for Blood Pressure). The Result Details window is displayed. The Results Detail window could be a graph of the results for determining trends. For a row like the Estimated Creatine Clearance, the Milliter a Minute could be on the Y axis and the Date could be on the X axis. The window could contain a chart of the results by date and time as well. If you want to mark a result as Reviewed or Signed and notify other parties, you can click the Forward button. The system displays the Forward pop-up window. Complete the requested information and click OK. The system returns you to the Result Details window.
        Note
        You can optionally print the content of the Results Detail window by clicking Print.
      2. To display the result trends from the Result Details window, click the underlined term for Trend. The system displays the Event window, which contains the Event Date, Event Name, Result, and the Normal Reference Range.
      3. To return to the MPages view, click the X in both the upper right corner of the Event window and the Result Details window (or click Close in this window).
    4. Problems and Diagnosis component - If a search box is displayed below the component header, enter a few characters of a nomenclature search term to display a matching list. Select the result you want from the drop down list, and the component data is displayed below it.
      Release Considerations
      The Quick Add Search functionality described above was added with Content Package 50522 (Inpatient Summary MPages View 2.3 (Feb 2011)). If you plan to use this functionality, make sure server 396 is running and the correct preferences are configured in Preference Maintenance Tool (PrefMaint.exe). For details on the functionality and the Preference Maintenance Tool configuration, see the Problems and Diagnosis design component pages.
      Release Considerations
      Beginning with MPages Content Package 51872 ( MPages Component Library 2.4 (May 2011)) and MPages Bedrock Configuration Content Package 51895 (MPages: Inpatient Summary Configuration 2.0 (May 2011)) and its replacement, the Significant Events component described below became available.
    5. Significant Events component - The Significant Events component alerts physicians, nurses, RRT team members, and Pharmacists to active significant clinical events that have occurred within the last 24 hours (default) for a patient. The triggering event depends on the type of Event (i.e. Sepsis Event, RRT Event, AKI Event) and Lighthouse offerings purchased by a client that contain specific Significant Event content. Rules are selected by a client within the Bedrock Wizard.
    6. New Order Entry component - You can sign results and review unsigned orders for tabs like Medications, Laboratory, Imaging, Billing, and other user defined areas. The following example describes how this component is used.
      1. Select a tab in the component like Labs. A list of the Laboratory categories is displayed below the component tab.
      2. Click a category row like CDC. The system displays the outstanding orders below the categories in a list.
      3. If you want to sign an order, select those you want from the list below the categories. For example, you could select multiple orders by clicking the box for Chest CT W/O Contrast, Chest XR 2 Views, Chest XR 1 View if they are displayed in the Orders for Signature list.
        Note
        You can click multiple items from every tab first, and then the Orders for Signature list would contain every order you selected from those tabs. This step would allow you to sign multiple orders.
      4. Click the Sign button. The system returns you to the MPages view.
        Note
        If you would prefer to review the unsigned orders before you sign those in the list, click the DX Table button. The system displays the Diagnosis Table for Unsigned Orders window. After you review the unsigned orders, click the Home button at the top of the window. You are returned to the MPages view. You can then sign those you want by selecting them from the list.
        Release Considerations
        Beginning with MPages Content Package XXXXX ( MPages Component Library 4.0 (July 2011)) and MPages Bedrock Configuration Content Package XXXXX, XXXXX, XXXXX, and XXXXX (MPages: XXXXX Summary Configuration 4.0 (July 2011)) and its replacement, the New Order Entry component described above became available.
    7. Documents component - You can update a PowerNote and view documented results using this component.
      1. Click a results row that contains a blue underlined link, which means there are documented results. The Document Viewer window is displayed.
      2. To update the PowerNote on the patient's chart, click the fourth icon over from the left on the toolbar in the Document Viewer window. The system launches the Open Note window.
      3. Select either Current Encounter or All Encounters on the Existing tab. Select the note you want from the list and click Make Correction. The system displays the Open Note editor where you can modify the note.
        Note
        The Open Note window contains tabs for Encounter Pathway, Existing, Precompleted, Catalog, Recent, and Favorites.
      4. Close the open windows to return to the MPages view.
        Release Considerations
        Beginning with MPages Content Package XXXXX ( MPages Component Library 4.0 (July 2011)) and MPages Bedrock Configuration Content Package XXXXX, XXXXX, XXXXX, and XXXXX (MPages: XXXXX Summary Configuration 4.0 (July 2011)) and its replacement, the Documents Results viewer enhancement described above became available.

Option Two:

  1. Navigate to the MPages Summary view.
  2. If it is not already expanded (you see a minus sign next to the component title), expand the component as described in Option One above.
  3. Position your mouse over a result row or column to display a pop-up window containing additional details. See the MPage components described below.
    Note
    These are just a few of those offered. For instructions on how to use some of the other components, see Option One above.
    1. Lines, Tubes, and Drains component - The units of output for a line, tube, or drain are displayed, and the documented date and time are displayed as well.
    2. Allergies component - The Allergy Name, Reaction, Severity, Status, Onset Date, and PowerChartÂ® comments are displayed.
    3. Labs component - The Normal Low, Normal High, Critical Low, and Critical High values are displayed for a category column like the Latest or Previous result.
