MPages Help Pages : Viewing and Updating Current Status Using Rehabilitation Summary
This page last changed on Jul 12, 2011 by JP3026.

Once you have displayed a patient and clicked the Rehabilitation Summary MPages tab in PowerChart, the Rehabilitation Summary MPages view is displayed. To view and update the current status and use the features provided by this component, complete the following steps:

Using the Ad-Hoc Charting Feature

  1. To document results by PowerChart category, click "+Add" on the Current Status component. The system launches the Ad-Hoc Charting window in PowerChart.
  2. Double-Click a charting category. The system populates the right side of the Ad-Hoc Charting window with a selectable list of items that relate to the category. For example, if you double-clicked Emergency Care as a charting category, the system would add selectable values for ED Abnormal Results Call To Patient, ED Call for Consults, ED Education and Teaching, ED Triage - Adult, etc.
  3. Select an item from the right-side of the window and click Chart. The system displays the charting details window.
    Note
    If you click Close, the system displays a message indicating you have selected one or more items to chart. It asks whether you want to chart the selected item and provides a Yes or No button. If you select Yes, the system displays the Details window. If you select No, you are returned to the Ad-Hoc Charting window.
  4. Fill-In the requested details, and click Save on the toolbar of the charting details window. The system adds the charting details into the patients electronic medical record in PowerChart, and you are returned to the MPages view.

Using the Drop Down Indicator Feature

  1. To update the patient's current status, click the blue triangle icon in the component header. The system displays a drop down list (menu) with selectable categories like Sleep Log,OT Inpatient Evaluation, and SLP Inpatient Progress Note.
  2. Select an item from the list. If you selected SLP Inpatient Progress Note, the PowerChart Review window is displayed. You can then update the progress notes by updating the date interval and whether you have reviewed the plans.
    Note
    You can add the subjective reason for treatment, the reason for referral, and the chief complaint if you select OT Inpatient Evaluation. You can update the sleep log if you select Sleep Log from the drop down list.
  3. When you have completed the progress note on the window, click the Save icon on the toolbar. The system closes the Review window and returns you to the Rehabilitation Summary MPages view.

Using the Date and Time Display for Assessment Categories

  1. Click any underlined row in the Current Status component like Peripheral Pulse Rate. The system displays The Peripheral Pulse Rate Details window showing a chart and a table of the pulse rate by date and time for every date and time a result was taken.
