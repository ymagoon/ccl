MPages Help Pages : Overview of Orthopedic Summary
This page last changed on Jul 12, 2011 by JP3026.

The Orthopedic Summary MPages view is focused on providing a patient summary view for hospital clinicians. Also, this page allows clinicians to take action in PowerChart directly from it. The Orthopedic Summary MPages view contains the following components:

Diagnoses, Vital Signs, Medications, Home Medications, Past Medical History, Diagnoses, Labs, Documents, Patient Information, New Order Entry, Procedure History, and Care Management (Discharge Planning).

You can hover over the section header of the Orthopedic Summary MPages view to get result details, or you can click the header to see the related patient information in PowerChart.

For more information see the All About Orthopedic Summary Reference Page.

Tasks

  • From PowerChart you can perform actions (for example, Mark as Reviewed, Add New, Modify, View, Remove, and so on.)
  • Within the Vital Signs component, besides the header taking you to the place in PowerChart where vitals are charted, each result displays a graphical image of the most recent results when you click a result value.
  • The Lab component graphs in this same manner, select the result, and it displays the most recent lab results in graph form.
  • By selecting the link in the Documents section, you can open the document directly from the Orthopedic Summary.
    To return to the Orthopedic Summary from the document viewer, select the Home icon at the top of the reader.
