MPages Help Pages : Viewing Notes and Documents in Nursing Communication
This page last changed on Aug 14, 2011 by JP3026.

To view notes from the Documents component in the Nursing Communication MPages view, complete the following steps:

1. Select the documented date link that displays next to the document name you want to view.
2. The system displays the note.

You can modify a note by entering your updates in the box provided. The system saves and displays your new comments.
