MPages Help Pages : Displaying and Viewing Components Using Pulmonary Summary
This page last changed on Aug 09, 2011 by JP3026.

There are a couple of ways for displaying and viewing information on the Pulmonary Summary MPage. You can follow one or more of the options to display the appropriate data.

Option One:

  1. Navigate to the MPage Summary view.
  2. If a "Add" is displayed to the right of the component title, select it to expand the component. Below are some examples describing the expanded component data.
    1. Lines, Tubes, and Drains component - "Add" expands the component window where the procedure and the documented date and time are displayed.
    2. Allergies component - "Add" expands the Allergies component, which lists each known drug and food allergy for a patient.
    3. Labs component - "Add" expands the Labs component where the result categories like CO2, BUN, and WBC are displayed. The latest and previous results are displayed under a column for the result category as well. A graph can be displayed for a result category when it has an underlined link. For example, if you click CO2 (it has a blue underlined link), the system displays a graph with the MEq/L (milliequivalents/liter) on the Y axis and the date and time on the X axis.
      Release Considerations
      Beginning with MPages Content Package XXXXX (MPages: Pulmonary Summary MPages View 2.4 (May 2011)) and its replacement, you can launch the Results viewer containing additional details by clicking a results column in the Labs component.
    4. Problems and Diagnosis component - If a search box is displayed below the component header, enter a few characters of a nomenclature search term to display a matching list. Select the result you want from the drop down list, and the component data is displayed below it.
      Release Considerations
      Beginning with MPages Content Package 53272 ( MPages Component Library 2.5 (August 2011)) and MPages Bedrock Configuration Content Package 53280 (MPages: Pulmonary Summary Configuration 2.0 (August 2011)) and its replacement, the quick add search functionality described above was added. If you plan to use this functionality, make sure server 396 is running and the correct preferences are configured in Preference Maintenance Tool (PrefMaint.exe). For details on the functionality and the Preference Maintenance Tool configuration, see the Problems and Diagnosis design component pages.
      Release Considerations
      Beginning with MPages Content Package 53272 ( MPages Component Library 2.5 (August 2011)) and its replacement, the Significant Events component described below became available.
    5. Significant Events component - The Significant Events component alerts physicians, nurses, RRT team members, and Pharmacists to active significant clinical events that have occurred within the last 24 hours (default) for a patient. The triggering event depends on the type of Event (i.e. Sepsis Event, RRT Event, AKI Event) and Lighthouse offerings purchased by a client that contain specific Significant Event content. Rules are selected by a client within the Bedrock Wizard.

Option Two:

  1. Navigate to the MPage Summary view.
  2. If it is not already expanded (you see a minus sign next to the component title), expand the component as described in Option One above.
  3. Position your mouse over a result row or column to display a pop-up window containing additional details. See the MPage components described below.
    Note
    These are just a few of those offered.
    1. Lines, Tubes, and Drains component - The units of output for a line, tube, or drain are displayed, and the documented date and time are displayed as well.
    2. Allergies component - The Allergy Name, Reaction, Severity, Status, Onset Date, and PowerChart® comments are displayed.
    3. Labs component - The Normal Low, Normal High, Critical Low, and Critical High values are displayed for a category column like the Latest or Previous result.
