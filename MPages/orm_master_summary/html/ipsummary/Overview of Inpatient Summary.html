MPages Help Pages : Overview of Inpatient Summary
This page last changed on Apr 12, 2010 by MB9512.

The Inpatient Summary MPage is focused on providing a patient summary view for hospital clinicians. Also, this page allows clinicians to take action directly from it.

The Inpatient Summary MPage contains the following components:
Patient Information, Diagnoses, Problems, Allergies, Medications, Home Medications, Vital Signs, Measurements and Weights, Intake and Output, Diagnostics, Microbiology, Labs, Pathology, Documents.

You can hover over the section header of the MPage to get result details or click on the header to be taken to the related patient information in PowerChart.

For more information see the All About Inpatient Summary Reference Page.

Tasks

  • From PowerChart you can perform actions (for example, Mark as Reviewed, Add New, Modify, View, Remove, and so on.)
  • Within the Vital Signs component, besides the header taking you to the place in PowerChartwhere vitals are charted, each result displays a graphical image of the most recent results when you click on the result.
  • The Lab component graphs in this same manner, select the result, and it displays the most recent lab results in graph form.
  • By selecting the link in the Documents section, you can open the document directly from the Inpatient Summary.
    To return to the Inpatient Summary Page from the document viewer, select the Home icon at the top of the reader.
