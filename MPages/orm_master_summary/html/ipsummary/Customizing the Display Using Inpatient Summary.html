MPages Help Pages : Customizing the Display Using Inpatient Summary
This page last changed on Sep 18, 2010 by JP3026.

You can rearrange the order that MPage components display and define the default behavior for each one. Complete the steps below to customize the component display:

  1. Click the Customize link in the upper right-hand corner of the MPage. The system displays the User Customization page where you can adjust the component behavior.
  2. Left-click a component and drag it to its new location or column.
    Note
    You can change the MPage layout to contain up to three columns. You change the layout from a three column view to a two column view by dragging and dropping the components into the appropriate number of columns. Components with a plus sign display as being collapsed, and components with a negative sign display as being expanded. To change the default behavior for a component, click the appropriate plus (+) or minus (-) sign. To clear the previously defined preferences and return to the default view defined in the Bedrock® wizard, click Clear Preferences. To expand or collapse every component when the MPage is displayed, click Expand All or Collapse All. When you position the mouse over a plus icon, the system should display "Expand". When you position the mouse over the minus icon the system should display "Collapse".
  3. When you have completed your modifications, click Save Preferences.
