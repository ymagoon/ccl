Overview

The New Order Entry component allows you to select orders from your favorites while reviewing data on the summary page, and then you can send them to the orders scratch pad to be signed. The list of available orders comes from your favorites in PowerOrders, and is automatically sorted by catalog type into the Meds, Labs, Imaging, Billing, and Other categories.

You can add multiple orders from the different categories by clicking on them and moving them to the scratch pad. If you change your mind about ordering one of them, uncheck the box next to that order. Once you are ready, click Submit for Signature and you are taken to PowerOrders where you can complete any required details and sign the order. Once the orders are signed, you are returned directly to the summary page. If you needed to place additional orders that are not in your favorites list, for example, you need to order an aspirin and an additional order you do not have on the list, you can check the box next to Search Mode and your orders still go to PowerOrders, but in addition the order search window is already open. Now you can search for additional orders, place those, and continue.

Clicking the section header takes you to PowerOrders, and clicking the Add button takes you to Orders with the Add Order window already opened.

Component Details

Heading

  • New Order Entry
  • The +Add button launchs the Add Orders window in PowerOrders

Content

  • Under each category, display user favorites
  • If no user favorites are defined for a category, display "No <order category> favorites found"
  • Categories - Meds, Labs, Imaging, Billing, Other

Actions

  • Click an order to add to the scratch pad.
  • Once an order is selected from the favorites list, change the font color to gray to indicate that it is not selectable once it has been added.
  • Allow the order to be removed from the scratch pad.
  • Allow for selection from all order categories.
  • Click Submit for Signature to open the order entry window.
  • The Search Mode check box will open the Add Order tab in search mode.
  • Once order details are completed and the order is signed, the order is removed from the scratch pad.

Hover

  • None at this time.
