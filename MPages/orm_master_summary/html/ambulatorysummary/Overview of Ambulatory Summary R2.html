Overview

The Ambulatory Summary MPage is focused on providing a patient summary view for ambulatory clinicians. Additional functionality allows clinicians to take action directly from the summary MPage. In phase one, the actions include the ability to add allergies, the ability to add documentation ( PowerForms, PowerNote), and the ability to add new orders using the Modal Order Entry window. Also, this phase supports user-based customization, which allows you to rearrange components on the page and to define the default expand/collapse behavior for each component.

The Ambulatory Summary MPage contains the following components:

Patient Information, Diagnoses, Problems, Allergies, Medications, Patient Histories including Past Medical, Procedure, Family History, Social History, Pregnancy History for female patients, Vital Signs and Measurements, General Labs, Diagnostics, Health Maintenance, Immunizations, Visits, Microbiology, Pathology, Documents, New Order Entry, Outstanding Orders, CDC Growth Chart, and Notes and Reminders.
