Overview

The Orders My Search component in the Ambulatory MPages view enables incremental and suggestion search that allows you to search for an order by mnemonic and order details.

My Search MPages Component

This component consists of the following sections:

  1. My Search Option
  2. Search and Results
  3. Filters

My Search Option

When you search for an orderable in My Search, qualifying results (synonyms and/or order sentences) display in a drop down list.

Search and Results

Enter an orderable in the My Search option to display qualifying synonyms and reference order sentences.

Note
The search does not return IV Sets, PowerPlans, Regimens, Continuous and IV Meds.
Note
The drop down displays 10 search results.

When you search for qualifying synonyms and order sentences, the results display based on the following criteria:

  • Left weighting for characters.
  • Not skipping for numbers.
  • Partial word matching.
  • Characters from multiple words separated by spaces.
  • Out of order word matching.
Note
Synonyms that are returned in the search display the synonym name only. Reference order sentences that are returned in the search display with the synonym name followed by the information that is displayed in the CDL.

Filters

The following filters are honored:

Note
Privilege Checking is completed when you sign an order.
  • Mnemonic
  • Catalog Type
  • Clinical Category
  • Virtual View (Facility)
  • Venue
  • Orderable Type Encounter Group

Using My Search

To search for an orderable and sign an order, follow these steps:

  1. Enter an orderable in My Search to display 10 qualifying results (synonym and/or order sentence) in a drop down list.
  2. Click on a synonym or order sentence. Once you click on a synonym, it is added to the Scratchpad.
  3. To remove an orderable from the Scratchpad, click on the orderable again.
  4. Click Submit for Signature to display the Modal Order Entry window (MOEW).
  5. If you click on an order sentence, it is added to the Scratchpad and you can sign it in the Ambulatory Summary MPages view.
  6. Once Order Entry has been completed (either in the Ambulatory Summary MPages view or in the MOEW, the order is removed from the Scratchpad and added to the Outstanding Orders and Medications components.
