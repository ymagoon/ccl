Overview

The Consolidated Problems component provides the ability for the user to quickly add problems and diagnoses. This also provides a complete view of the charts problem status to allow providers to manage problems effectively.

Adding a Term to This Visit Section

To add a term to This Visit, you can select any of these options:

  • Select This Visit from the list, search for and select a term in the search box.
  • Select This Visit and Active from the list, search for and select a term in the search box.
  • Select a condition from the Active section, drag and drop it to This Visit section.
  • Select a condition from the Historical section, drag and drop it to This Visit section.
  • Select a condition from the Active section and select Move to This Visit option from the action menu.
  • Select a condition from the Historical section and select Move to This Visit option from the action menu.
Note
Adding a term to the This Visit section creates a diagnosis on the current chart encounter.

Adding a Term to Active Section

To add a term to the Active section, you can select any of these options:

  • Select This Visit and Active from the list, search for and select a term in the search box.
  • Select Active from the list, search for and select a term in the search box.
  • Select a condition from the This Visit section, drag and drop it to the Active section.
  • Select a condition from the Historical section, drag and drop it to the Active section.
  • Select a condition from the This Visit section and select the Move to Active option from the action menu.
  • Select a condition from the Historical section and select the Move to Active option from the action menu.
Note
Adding a term to the Active section adds an active problem to the patient. When a condition is selected that contains a problem and a drag and drop action is taken or the Move to Active option is selected, the problem status is updated to Active.

Adding a Term to Historical Section

To add a term to the Historical section, select Historical from the list, then search for and select a term in the search box.

Note
Adding a term to the Historical section adds a resolved problem to the patient.

Changing Status of a Problem

To change the status of a problem, complete the following steps:

  1. Select a condition that contains a problem in any section and select Resolve, Inactivate or Cancel from the Action menu.
    Note
    These three options are available depending on the current status of the problem. For example, if the problem is already in a Resolved status, the Resolve menu option is not available.
  2. Drag and drop a condition that contains a problem from either the This Visit section or the Historical section to the Active section to update the status of the problem to Active.

Filtering by Classification

To filter by classification, you can select any of the following options:

  • From the Classification filter, select All to view all the problems and diagnoses in the list regardless of their classification.
  • From the Classification filter, select Medical and Patient Stated to filter out all problems and diagnoses that do not have a classification of Medical or Patient Stated.

Removing a Condition from This Visit Section

To remove a condition from the This Visit section, select a condition in the This Visit section and select Remove From This Visit from the Action menu.

Note
When you remove a condition from the This Visit section, you are inactivating a diagnosis.

Viewing Details About a Condition

To view details about a condition, complete the following steps:

  1. Click + next to the condition to view comments.
  2. Position the cursor over the condition to view details.
