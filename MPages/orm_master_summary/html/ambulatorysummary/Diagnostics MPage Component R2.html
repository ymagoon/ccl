Overview

The Diagnostics component displays all diagnostic results for the patient within the last six months. The time frame is displayed in the component heading. Results are grouped under three sub-headings: EKG, Chest/Abdomen X-Rays, and Other Radiology. Each site determines what specific tests display under each heading. Items are sorted in reverse chronological order by resulted date. The Within column displays the amount of time elapsed since the study was completed, but positioning the mouse over the time frame displays the study name and the actual date and time the study was completed.

You can click the time frame to see the full results of the study. From this page you can click the back arrow button or the home button to return to the Ambulatory Summary MPage.

Component Details

Heading

  • Diagnostics (n) Last six months
    n = total number of items displayed
  • Component heading will link to appropriate component in the chart

Layout

  • First Column - result name
  • Second Column - time elapsed since result was documented, hyperlink to the result
  • Third Column - status

Content

  • EKG
    • If no results are documented for this category, it will display "No results found"
    • Modifications: The Bedrock Wizard allows you to define the EKG results to display in this grouping.
  • Chest / Abd XR
    • If no results are documented for this category, it will display "No results found"
    • Modifications: The Bedrock Wizard allows you to define the chest / abdomincal x-ray results to display in this grouping.
  • Other Diagnostics
    • If no results are documented for this category, it will display "No results found"
    • Modifications: The Bedrock Wizard allows you to define other diagnostic results to display in this grouping.

Data Format

  • Look Back Range: six months
  • Status: all resulted diagnostics
  • Sort Order: date documented in reverse chronological order
  • Truncate / Wrap: text will wrap if it exceeds the column width
  • Scrolling: client defined
  • Date/Time Display: listed as elapsed time since result was documented
  • Date/Time of the result will link to the appropriate diagnostic report

Hover

  • Study Name
  • Date/time the study was completed
