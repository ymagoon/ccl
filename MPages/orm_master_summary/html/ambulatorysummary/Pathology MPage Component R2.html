Overview

The Pathology component displays the pathology results from the last two years for a patient. Results are displayed in reverse chronological order. Each site determines the specific note types that display in this component.

For any pathology results, you can click the date to see full details of the report. From here you can click the back arrow or home button to return to the summary page.

Component Details

Heading

  • Pathology (n) Last 2 years
    n = total number of pathology items
  • Component heading will link to appropriate component in the chart

Content

  • Name of the Report
  • Date (link to the report)

Data Format

  • Look Back Range: last 2 years
  • Status: displays Auth, Altered, and Modified reports
  • Sort Order: display results in reverse chronological order based on date / time
  • Truncate / Wrap: text will wrap if it exceeds the column width
  • Scrolling: client defined
  • Date / Time: date of the result will link to the appropriate report

  • None at this time
