MPages Help Pages : General Labs MPage Component R2
This page last changed on Aug 17, 2010 by RP018820.

The Labs component displays lab results that are selected by your site. For a given lab item, it displays the three most recent results, looking back up to 2 years. Your site may define a different amount of time for the look back range. That time frame is displayed in the section header. Positioning the mouse over a result displays the normal range for that result, the unit of measure, and the date.

Clicking the result name displays a graph containing all the data points for the last two years for that item. Clicking the section header takes you to the Flowsheet in the chart.

Component Details

Heading

  • Labs
  • Last x days, weeks, months, or years
    x = client defined time frame
  • Component heading will link to appropriate component in the chart

Layout

  • First Column - lab result name
  • Second Column - latest result
  • Display a vertical colored line separator between latest and previous results
  • Third and Fourth Columns - previous results

Content

  • Client can define what labs display and what sequence

Out of Range Results

  • Critical High - bold red font, red exclamation point icon
  • Abnormal High - bold orange font, orange up arrow icon
  • Abnormal Low - bold blue font, blue down arrow icon

Data Format

  • Look Back Range: client defined (default = 2 years)
  • Status: three most current results
  • Date/Time Display: elapsed time since the result was documented, under each result
  • Sort Order: client defined

Graphing

  • Display the result label as a hyperlink to the graph view
  • Graph will display all data points for the result in the last 2 years
  • Hover over each data point to display the actual result, unit of measure, and date/time resulted
  • Display a table of results to the right of the graph, sorted in chronological order

Hover

  • Result
  • Normal Range
  • Unit of Measure
  • Date Time
