Overview

The Documents component displays all documents completed on the patient for the previous two years. Each site determines what note types are displayed. Notes are sorted in reverse chronological order by the date and time. Hovering over a note displays the note type name, subject, status, and date and time.

Click the document date to view the document. Click the back arrow button or home button when you are ready to return to the summary page. Clicking the Documents section header takes you to the Document Viewing component in the chart. You also have a couple options to start a new note directly from the summary screen. You can click Add which takes you to Document Viewing with the new note dialog started. You can also select one of your PowerNote favorites from the drop down list to open the note directly.

After documenting a couple items, sign the note and close it. The note you just completed displays on the summary screen.

Component Details

Heading

  • Documents (n)
    n = total number of documents displayed
  • + Add button that will open the new note dialog in Document Viewing
  • Drop down list (blue arrow) of the user's PowerNote favorites that can be launched
  • Component heading will link to appropriate component in the chart

Content

  • Note Type (hyperlink to the note)
  • Subject
  • Author
  • Date

Data Format

  • Look Back Range: last two years
  • Note Types to Display: client-defined
  • Status: all statuses except In Error
  • Sort Order: documents listed in reverse chronological order by result date/time
  • Scrolling: client defined
  • Truncate / Wrap: text wraps if it exceeds the column width

Hover

  • Display label "Name": followed by the note type.
  • Display label Subject: followed by the subject of the document.
  • Display label "Status": followed by the document status.
  • Display label "Date/Time": followed by the last update date and time.
