Overview

The Notes and Reminders component displays all active sticky notes and reminders for the patient. The component pulls the Sticky Notes into the view from the Sticky Note functionality on the PowerChart Toolbar if enabled. There is a Sticky Note sub-heading and a Reminder sub-heading. Items are sorted in chronological order under both. You can click on the Reminder hyperlink to see the actual message. Clicking the section header takes you to the component in the chart determined by your site.

Component Details

Heading

  • Notes/Reminders (n) Active
    n = total combined number of active notes and reminders
  • Component heading will link to appropriate component in the chart

Content

  • Sticky Note - Author, Date, Text
  • Reminder - Priority, Subject, Show Up Date/Time, Due Date/Time, Status

Data Format

  • Look Back Range: lifetime
  • Status: all active
  • Sort Order: chronological
  • Truncate / Wrap: text wraps if it exceeds the column width
  • Scrolling: client-defined

Hover

  • None at this time
