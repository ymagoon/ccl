C. controllers

Array of controller names to add to the application definition.

example

C.controllers.push("Perm")
Ext.define('MPAGE.controller.Perm', {
    extend: 'Ext.app.Controller',
    ...
})
...
Ext.application({
    name: 'MPAGE',
    controllers:C.controllers,
    ...
})
