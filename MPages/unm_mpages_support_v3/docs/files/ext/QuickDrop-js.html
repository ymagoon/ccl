univnm. ext. QuickDrop

A simple combobox implementation for static lists

Detail

This component takes all the normal ComboBox config options, except

  • store
  • valueField
  • displayField
  • mode

which are set internally.  Instead you set a values property that is a JS object of label:value pairs

Example

...
xtype:"quickdrop",
name:"is_awesome",
fieldLabel:"Are you Awesome?",
values:{
    "Hell Ya!":1,
    "um, what?":0
}

...

You can also pass an array of objects

...
xtype:"quickdrop",
name:"is_awesome",
fieldLabel:"Are you Awesome?",
values:[
    {label:"Hell Ya!", value:1},
    {label:"um, what?",value:0}
]

...

.. or just an array of values

...
xtype:"quickdrop",
name:"status",
fieldLabel:"Status",
values:["ACTIVE","INACTIVE","DENIED"]

...
