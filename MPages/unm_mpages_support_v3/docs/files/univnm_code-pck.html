univnm_code_code

University of New Mexico’s PL/SQL library to support custom tables and CCL-to-SQL gateway

This package provides the following public functions

  • code_lookup
  • code_display
  • code_display_key

These are intended to be used from queries

by_display_key

function by_display_key (code_set_in IN number,
display_key_in IN varchar2) return number

returns code by codeset and display key, like uar_get_code_by(“DISPLAYKEY”,...)

Parameters

code_set_inIN number.  Code set to search
display_key_inIN varchar2.  Display key to search

Example

select *
from orders s
where
   s.order_status_cd != univnm_code.by_display_key(6004, 'CANCELED')

Note

in the case of no match found, -1 is returned
