Run Queries:

Select the "New Query Tab" button to open a new Query Window, or open a table i nthe left-had grid, Right-click on a column name and select "Query Table"

 

Insert SQL:

Drag and drop a column name to insert. A menu will appear offering to insert the table name, column name, a join or a list of all the column names

 

Query Parameters:

To create query parameters, add curly placeholders to you SQL, e.g {user_id}, {node}, etc. This will be picked up added to the "Variables" grid. Any values entered into the Variables grid will be merged into the SQL when running a query
