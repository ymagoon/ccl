R1 MPages Help Pages : Adding a Pregnancy
This page last changed on May 25, 2010 by JS015791.

Within the patient chart in PowerChart(Powerchart.exe), select the Table of Contents row containing the Pregnancy Summary mPage.  If the patient does not have a current active pregnancy documented, the system displays the following message: There are currently no active pregnancies for this patient.

Complete the following steps to add a pregnancy to your patient:

  1. Click the Add Pregnancy button. The Add Pregnancy window opens.
  2. Document onset date and time, as well as any other pertinent information. 
  3. Enter any comments you would like recorded.
  4. Click OK.

Once you have added the pregnancy information, the system displays the Pregnancy mPage. 

Note:
The result timeline component of the mPage is driven by the pregnancy onset date and time.
