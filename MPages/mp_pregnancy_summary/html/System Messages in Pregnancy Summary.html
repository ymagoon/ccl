R1 MPages Help Pages : System Messages in Pregnancy Summary
This page last changed on Apr 12, 2010 by MB9512.

The system messages listed below could be displayed in Pregnancy Summary. The information displayed contains an explanation of the message and the action you should take to correct the issue.

If appropriate, a more technical description is also provided for use by your system administrator. If the technical description mentions contacting Cerner to correct the problem, it will be very helpful if you provide the following information to your system administrator:

  • The details of what happened just before the error occurred.
  • Any information that you may have entered.
  • The date and time the error occurred.
  • If you receive an Illegal Operation... error, click Details and then take a screen shot (pressing ALT+PRINT SCREEN is one method) of the information.

The messages shown below are sorted first by numbers, if the message includes one; then by symbols, such as brackets ([ ]), which enclose message text that is variable; and finally, alphabetically, by the message text itself.

Messages

There are currently no active pregnancies for this patient.
Explanation: There are currently no active pregnancies in the system for this female patient. Add a pregnancy to display the MPage Pregnancy Summary components.
Action: Select Add Pregnancy or exit.

You must select a female patient to view the Pregnancy Summary
Explanation: You cannot create a pregnancy summary for a patient with a male gender. Verify you have the correct patient and patient gender.
Action: Exit the window.
