R1 MPages Help Pages : Adding an EDD Maintenance Module in Pregnancy Summary
This page last changed on May 25, 2010 by JS015791.

 

Open the patient chart to the Pregnancy Summary mPage of a patient with an active pregnancy.  

To add an Estimated Due Date (EDD) using the EDD Maintenance Module complete the following steps:

  1. On the current pregnancy tab within the mPage, the system displays a hyperlink text of Add EDD if the patient does not have an EDD already documented.
  2. Select Add EDD.
  3. The EDD Maintenance Tool opens in Add mode.
  4. Select the method by which the EDD was calculated (last menstrual period, ultrasound, or advanced reproductive technology) and the related details.  The associated details vary based on the EDD calculation method selected. 
  5. Select whether the EDD is initial, final or authoritative.
    Note:
    The system uses the status when making EDD calculations.  The EDD can be updated/reassessed due to subsequent ultrasounds.
  6. Click OK to complete the addition of the EDD.
