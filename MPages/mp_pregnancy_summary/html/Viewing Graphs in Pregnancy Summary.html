R1 MPages Help Pages : Viewing Graphs in Pregnancy Summary
This page last changed on Apr 12, 2010 by MB9512.

Assessments

Complete the following steps to view patient information displayed in the Assessments component in graph format:

  1. Select the drop-down arrow in the View Graph spin box.
  2. Select Fundal Height from the list provided to see the win32 Fundal Height Graph.
  3. Select Labor Graph from the list provided to see the win32 Inpatient Partogram.
  4. Click Print to print a copy of the graph displayed.
  5. Click close to exit the window.

Labs
In the Labs component of the Pregnancy Summary Mpage there are lab results listed as links. To view laboratory patient information in graph formatting select a link you want to view. The Mpage will open the results as a graph. Close the graph when you have finished reviewing.
