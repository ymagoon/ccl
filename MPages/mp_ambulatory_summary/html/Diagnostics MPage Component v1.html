R1 MPages Help Pages : Diagnostics MPage Component v1
This page last changed on Jan 15, 2010 by SM015827.

The Diagnostics component displays all diagnostic results for the patient. Each site can determine how far back in time the component will pull results. The time frame is displayed in the component heading, and in this case it is looking at the last six months. Results are grouped under three sub-headings: EKG, Chest/Abdomen X-Rays, and Other Radiology. Each site will determine what specific tests display under each heading. Items are sorted in reverse chronological order by resulted date. The Within column displays the amount of time elapsed since the study was completed, but positioning the mouse over the time frame will display the study name and the actual date and time the study was completed.

I can click the time frame to see the full results of the study. From this page I can click the back arrow button or the home button to return to the summary page.

Component Details

Heading

  • Diagnostics (n) Last x (days, months, years)
    n = total number of items displayed
    x = look back range selected in the Bedrock Wizard
  • Component heading will link to appropriate component in the chart

Layout

  • First Column - result name
  • Second Column - time elapsed since result was documented, hyperlink to the result
  • Third Column - status

Content

  • EKG - site can define the EKG results to display in this grouping
  • Chest/Abd XR - site can define the chest/abdominal x-ray results to display in this grouping
  • Other Diagnostics - site can define the other diagnostic results to display in this grouping

Data Format

  • Look Back Range: client defined (not encounter specific)
  • Status: all resulted diagnostics
  • Sort Order: date documented in reverse chronological order
  • Truncate / Wrap: text will wrap if it exceeds the column width
  • Scrolling: client defined
  • Date/Time Display: listed as elapsed time since result was documented
  • Date/Time of the result will link to the appropriate diagnostic report

Hover

  • Study Name
  • Date/time the study was completed
