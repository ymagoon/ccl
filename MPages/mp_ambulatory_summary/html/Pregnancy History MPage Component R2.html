R2 MPages Help Pages : Pregnancy History MPage Component R2
This page last changed on Aug 17, 2010 by RP018820.

The Pregnancy History component only displays in the summary page for female patients. It displays all pregnancy history records for the lifetime of the patient and pregnancy items which are sorted in chronological order. Hovering over an item displays length of labor, delivery hospital, child's name, and father of the baby's name.

Clicking the section header takes you to Histories in the chart, where you can access the Pregnancy History control for more information.

Component Details

Requirements
Use of the Pregnancy History component requires a license for PowerChart Maternity

Heading

  • Pregnancy(n)
    n = total number of pregnancy history items
  • Component heading links to appropriate component in the chart

Content

  • Display sub-categories of Pregnancy #1, Pregnancy #2, etc.
  • Under Pregnancy heading, display "Baby 1:" followed by date of birth, neonate outcome, pregnancy outcome/result, baby's gender, baby's birth weight and gestation at birth (wks and days). Display addition baby information rows as necessary.

Data Format

  • Look Back Range: lifetime
  • Sort Order: chronological order by pregnancy (pregnancy 1, pregnancy 2, etc.)
  • Truncate / Wrap: text wraps if it exceeds the column width
  • Scrolling: client-defined

Hover

  • Length of Labor: followed by length of labor in hours and minutes.
  • Delivery Hospital: followed by facility where delivery occurred.
  • Child's Name: followed by name of baby.
  • FOB Name: followed by the name of the baby's father.
