R1 MPages Help Pages : Past Medical History MPage Component v1
This page last changed on Jan 15, 2010 by SM015827.

The Past Medical History component displays all resolved problems for the lifetime of the patient. If there is an associated ICD-9 (or SNO-MED, ICD-10, etc.) code it will display in parenthesis next to the name.

Problems are sorted in alphabetically. (Video will be updated shortly.)

Positioning the mouse over a problem will display the problem name, onset date, resolved date, status and any comments. Clicking the section header will take you to the Past Medical History control in the chart. Notice that the MPage only displayed the appendicitis problem, but the Past Medical History control is displaying appendicitis and hypertension. That is because the PMH section of the MPage only displays resolved problems. The Hypertension problem is still active, but was filed to Past Medical History during entry. I can change my filter to resolved and now the display matches the MPage.

I'll click Add, go through my favorite folders, and enter a problem of back pain. It started at age 35 and resolved at age 36. Click OK. Now I'll go back to the Ambulatory Summary by clicking the Table of Contents. Refresh the screen, and you can see the back pain is now showing in my Past Medical History section.
Component Details

Heading

  • Past Medical (n)
    n = total number of past medical history items
  • Component heading will link to appropriate component in the chart

Content

  • Display problem name
  • Display associated code in parenthesis

Data Format

  • Look Back Range: lifetime
  • Status: display active and resolved problems
  • Sort Order: alphabetical
  • Truncate / Wrap: text will wrap if it exceeds the column width
  • Scrolling: client defined

Hover

  • Problem Name
  • Annotated Display
  • Onset Date
  • Resolved Date
  • Status
  • Comments
