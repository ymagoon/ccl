The Procedure History component displays all procedures for the lifetime of the patient. The year of the procedure is displayed to the right of the name.
Procedures are sorted alphabetically. (Video will be updated shortly.)

Positioning the mouse over a procedure will display the procedure name, date, status, provider, location, and any comments. Clicking the section header will take you to the History control in the chart; I can then click on the Procedure History tab. To add a procedure I'll click Add. From my favorite folders I want to add an IV therapy to go along with my appendectomy. That was at the age off 32. I'll return to the Ambulatory Summary by clicking in the Table of Contents, refresh the screen, and my procedure is now displaying.
Component Details

Heading

  • Procedures (n)
    n = total number of procedure history items
  • Component heading will link to appropriate component in the chart

Content

  • Display procedure name
  • Display date of procedure

Data Format

  • Look Back Range: lifetime
  • Status: display all active procedure history items
  • Sort Order: alphabetical
  • Truncate / Wrap: text will wrap if it exceeds the column width
  • Scrolling: client defined

Hover

  • Procedure Name
  • Procedure Date
  • Status
  • Provider
  • Location
  • Comments
