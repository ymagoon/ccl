The Immunization component displays all administered and historical immunizations for the lifetime of the patient. Items are sorted in reverse chronological order by the date given. Positioning the mouse over an item will display the immunization name, type, product, admin date, provider, details, manufacturer, lot number, expiration date, and admin notes.

Clicking the section header will take you to the Immunization Schedule in the chart. From here I'm going to document an historical immunization for this patient. <enter data>. Now I'll return to the summary page by clicking in the Table of Contents. I'll click refresh, and now I can see the immunization I just documented.
Component Details

Heading

  • Immunizations (n)
    n = number of immunizations given
  • Component heading will link to appropriate component in the chart

Content

  • Display immunization name
  • Display date given

Data Format

  • Look Back Range: lifetime
  • Status: all administered and historical immunizations
  • Sort Order: reverse chronological by date given
  • Truncate / Wrap: text will wrap if it exceeds the column width

Hover

  • Immunization
  • Type
  • Product
  • Admin Date
  • Provider
  • Details
  • Manufacturer
  • Lot #
  • Expiration Date
  • Admin Notes
