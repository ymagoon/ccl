R1 MPages Help Pages : Pregnancy History MPage Component v1
This page last changed on Jan 15, 2010 by SM015827.

The Pregnancy History component will only display in the summary page for female patients. It displays all pregnancy history records for the lifetime of the patient. Pregnancy items are sorted in chronological order. Hovering over an item will display length of labor, delivery hospital, child's name, and father of the baby's name.

Clicking the section header will take you to Histories in the chart, where you can access the Pregnancy History control for more information.
Component Details

Requirements
Use of the Pregnancy History component requires a license for PowerChart Maternity

Heading

  • Pregnancy(n)
    n = total number of pregnancy history items
  • Component heading will link to appropriate component in the chart

Content

  • Display sub-categories of Pregnancy #1, Pregnancy #2, etc.
  • Below each pregnancy display Baby #: followed by date of birth, neonate outcome, pregnancy outcome/result, baby's gender, birth weight, and gestation at birth

Data Format

  • Look Back Range: lifetime
  • Sort Order: chronological order by pregnancy (pregnancy 1, pregnancy 2, etc.)
  • Truncate / Wrap: text will wrap if it exceeds the column width
  • Scrolling: client defined

Hover

  • Length of Labor
  • Delivery Hospital
  • Child's Name
  • Father's Name
