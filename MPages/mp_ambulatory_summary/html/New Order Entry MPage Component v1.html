R1 MPages Help Pages : New Order Entry MPage Component v1
This page last changed on Jan 15, 2010 by SM015827.

The New Order Entry component allows you to select orders from your favorites while reviewing data on the summary page, and then send them to the orders scratch pad to be signed. The list of available orders comes from your favorites in PowerOrders, and is automatically sorted by catalog type into the Meds, Labs, Imaging, Billing, and Other categories.

I can add multiple orders from the different categories by clicking on them and moving them to the scratch pad. If I change my mind about ordering one of them, I just uncheck the box next to that order. Once I'm ready I click Submit for Signature and I'm taken to PowerOrders where I can complete any required details and sign the order. Once the orders are signed, I'm returned directly to the summary page. If I needed to place additional orders that aren't in my favorites list, say I need to order an aspirin and an additional order I don't have on the list, I can check the box next to Search Mode and my orders will still go to PowerOrders, but in addition the order search window will already be opened. Now I can search for additional orders, place those, and continue.

Clicking the section header will take me to PowerOrders, and clicking the Add button will take me to Orders with the add order window already opened.
Component Details

Heading

  • New Order Entry
  • +Add
  • The +Add button will launch the add orders window in the PowerOrders

Content

  • Under each category, display user favorites
  • If no user favorites are defined for a category, display "No <order category> favorites found"
  • Categories - Meds, Labs, Imaging, Billing, Other

Actions

  • Click an order to add to the scratch pad.
  • Once an order is selected from the favorites list, change the font color to gray to indicate that it is not selectable once it has been added.
  • Allow the order to be removed from the scratch pad.
  • Allow for selection from all order categories.
  • Click Submit for Signature to open the order entry window.
  • The Search Mode check box will open the Add Order tab in search mode.
  • Once order details are completed and the order is signed, the order is removed from the scratch pad.

Hover

  • None at this time
