R1 MPages Help Pages : Notes and Reminders MPage Component v1
This page last changed on Jan 15, 2010 by SM015827.

The Notes and Reminders component displays all active sticky notes and reminders for the patient. There is a Sticky Note sub-heading and a Reminder sub-heading. Items are sorted in chronological order under both. I can click on the Reminder hyperlink to see the actual message. Clicking the section header will take you to the component in the chart determined by your site.

As a demonstration I'm going to add a new reminder for myself by using the Communicate button. I'll make it a high priority, and I want to tell myself to follow up on some results. I'll have it due in one week. I send the message, refresh the screen, and now you can see the new reminder.
Component Details

Heading

  • Notes/Reminders (n) Active
    n = total combined number of active notes and reminders
  • Component heading will link to appropriate component in the chart

Content

  • Sticky Note - Author, Date, Text
  • Reminder - Priority, Subject, Show Up Date/Time, Due Date/Time, Status

Data Format

  • Look Back Range: lifetime
  • Status: all active
  • Sort Order: chronological
  • Truncate / Wrap: text will wrap if it exceeds the column width
  • Scrolling: client defined

Hover

  • None at this time
