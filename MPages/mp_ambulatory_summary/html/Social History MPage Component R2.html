Overview

The Social History component displays all social history items for the lifetime of the patient sorted alphabetically by category.
Hovering over an item displays the name of the category and any details that were documented.

Clicking the section header takes you to the History control in the chart. From here you can select the Social History tab.

Component Details

Heading

  • Social(n)
    n = total number of social history items
  • Component heading will link to appropriate component in the chart

Content

  • Display category name followed by the documented assessment

Data Format

  • Look Back Range: lifetime
  • Sort Order: by collating sequence and then alphabetical by category
  • Truncate / Wrap: text will wrap if it exceeds the column width
  • Scrolling: client defined

Hover

  • Category: followed by the name of the category.
  • Details: followed by the detail of the social history.
  • Last Updated: followed by the date the assessment was last updated.
  • Last Updated By: followed by the name of the provider.
  • Comments: followed by all associated comments about the social history.
