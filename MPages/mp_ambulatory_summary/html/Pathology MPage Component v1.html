The Pathology component will display the five most recent pathology results from the last 6 months. Results are displayed in reverse chronological order. Each site will determine the specific note types that will display in this component. For this demonstration I don't have any actual pathology results, so I have set the component up to display other note types that you wouldn't normally see in the Pathology component. For these notes, or any actual pathology result, I can click the verified date to see full details of the report. From here I can click the back arrow or home button to return to the summary page.
Component Details

Heading

  • Pathology (n) Last 5 results
    n = total number of pathology items
  • Component heading will link to appropriate component in the chart

Content

  • Name of the Report
  • Verified Date (link to the report)

Data Format

  • Look Back Range: last six months
  • Number of Results: five most recent results
  • Status: all statuses except InErrored
  • Sort Order: display results in reverse chronological order based on verified date / time
  • Truncate / Wrap: text will wrap if it exceeds the column width
  • Scrolling: client defined
  • Date / Time: display time elapsed since pathology report was verified
  • Date / Time: date of the result will link to the appropriate diagnostic report

Hover

  • None at this time
