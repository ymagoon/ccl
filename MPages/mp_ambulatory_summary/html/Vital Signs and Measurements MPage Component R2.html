Overview

The Vital Signs and Measurements component displays configured vital sign results in an order defined by your site. Hovering over a result displays the normal range, unit of measure, and the date and time the result was entered.

Clicking a result label displays the graph view for that result. The graph displays all data points for the last two years. In the graph view, you can hover over a data point to see the actual result, unit of measure, and date and time it was entered. There is also a table view of the results on the right side. The results in the table are in chronological order.

Close the graph to return to the main summary. Clicking the section header takes you to the Flowsheet section of the chart. You can also quickly document new results from the MPage by clicking Add and launching the Ad Hoc Charting window. You can also use the drop down list to select from a site defined list of PowerForms that can be launched by clicking on the name of the form.
Component Details

Heading

  • Vitals and Measurements with additional text to the right of the label based on the look back range selected. The options include:
    • Last n days
    • Last n weeks
    • Last n months
    • Last n years
  • Display "Last 2 years" to the right of the heading
  • Component heading links to appropriate component in the chart
  • +Add launches the Ad Hoc charting window
  • Drop down (blue arrow) displays client defined list of PowerForms

Layout

  • First Column - vital sign result name
  • Second Column - most recent result
  • Display a vertical colored line separator between the most recent and previous results
  • Third and Fourth Columns - previous results

Content

  • Display Temp (temperature), BP (blood pressure), HR (heart rate), and other client defined clinical events
  • Only display row if a result has been charted
  • Display out-of-range results in appropriate color and with abnormal indicator

Out of Range Results

  • Critical High - bold red font with red exclamation point icon
  • Abnormal High - bold orange font with orange up arrow icon
  • Abnormal Low - bold blue font with blue down arrow icon

Data Format

  • Look Back Range: client defined (default = last two years)
  • Data Available: display three most current results for each item
  • Sort Order: client defined
  • Date / Time Display: display the elapsed time since the result was documented, beneath the result as "x time ago"

Graphing

  • Display the result label as a hyperlink.
    Note
    You can click on the result label to display the graph view in a modal window.
  • Clicking on the result label displays the graph view.
  • Graph displays all data points for the last two years up to a maximum of 100 results.
  • Hover over data points to display the actual result, unit of measure, and date/time.
  • Display table of results to the right of the graph.
  • Display table column headings as "Date" and "unit of measure appropriate for the result".
  • Display results in chronological order.
  • You can zoom in and out.
  • You can print the current view of the graph including the fully expanded table of results.

Hover
When you hovers over an item in the list this information displays:

  • Result: followed by the actual result. This field will display any alpha symbols that have been ensured with numeric values, such as > or < om the "Result" field.
  • Normal Range: followed by the normal reference range for the result.
  • Unit of Measure: followed by the unit of measure for the result.
  • Date/Time: followed by the date and time of the result.
