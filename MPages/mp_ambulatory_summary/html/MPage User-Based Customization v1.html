In the Ambulatory Summary MPage the end user is able to rearrange components on the page and define the expand/collapse default for each component.  To do this, I'll click the Customize View link in the upper right-hand corner.  From here I simply left-click a component and drag it to it's new location.  I can change my layout from a three column view to a two column view by moving everything over into the first two columns.  Components that have a plus sign will default to being collapsed, and components with a negative sign will default to being expanded.  I can change the default by clicking on the icon.  When I'm finished, I click on Save Preferences and I'm taken back to the summary screen with my new layout.
