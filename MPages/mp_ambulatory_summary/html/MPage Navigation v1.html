Every individual component has an expand/collapse icon in the upper right corner. Components can default to being expanded like Vitals and Measurements is here, or default to being collapsed like Patient Information is. This default behavior can be customized at the user level as I show in the user-based customization help video. I can click on the icon to collapse and expand the particular component for the current session.

Moving up the navigation bar at the top, the first option is a search on the MPage. I can open the search by clicking ctrl+F on my keyboard, or clicking the binocular icon. As an example I'll search for Metformin, click Next, and I'm taken to the first instance of Metformin in the Medication section.

The next option to demonstrate is the back, forward, and home icons. If I navigate to another screen within the MPage, for example I'll view the latest office/clinic note for this patient, I can always click the home icon to return to the main summary screen. I can use the back and forward arrows to return to pages I've already visited.

The last thing I want to show is the collapse all / expand all option. In the upper right hand corner of the screen I can click collapse all to collapse everything on the summary page, and then click expand all to expand everything open again.
