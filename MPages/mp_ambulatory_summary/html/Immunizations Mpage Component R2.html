Overview

The Immunization component displays all administered and historical immunizations for the lifetime of the patient. Items are sorted in reverse chronological order by the date given. Hovering over an item displays the immunization name, type, product, admin date, provider, details, manufacturer, lot number, expiration date, and admin notes.

Clicking the section header takes you to the Immunization Schedule in the chart. Return to the Ambulatory Summary MPage by clicking in the Table of Contents.

Component Details

Heading

  • Immunizations (n)
    n = number of immunizations given
  • Component heading links to appropriate component in the chart

Content

  • Display immunization name
  • Display date given

Data Format

  • Look back range: Lifetime.
  • Truncate/Wrap: The text wraps if it exceeds the width of the column.
  • Result status: All charted immunizations in a done status will display.
  • Sort order: The immunizations will sort in reverse chronological order by given date.
  • Scrolling: Client defined.

Hover

  • Immunization: followed by the name of the immunization, vaccination, etc.
  • Type: followed by the type of immunization/vaccine.
  • Product: followed by the order mnemonic.
  • Admin Date: followed by the date the immunization was administered.
  • Provider: followed by the name of the provider who administered the immunization/vaccine.
  • Details: followed by the dose, unit, route, and site of the immunization/vaccine.
  • Manufacturer: followed by the name of the manufacturer of the immunization/vaccine.
  • Lot: followed by the lot number of the immunization/vaccine.
  • Expiration Date: followed by the expiration date of the immunization/vaccine.
  • Admin Notes: followed by any notes pertaining to the immunization/vaccine.
