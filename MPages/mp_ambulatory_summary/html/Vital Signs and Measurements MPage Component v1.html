R1 MPages Help Pages : Vital Signs and Measurements MPage Component v1
This page last changed on Jan 15, 2010 by SM015827.

The Vital Signs component displays a site defined list of vital signs and measurements. The three most recent results for each item will be displayed, looking back a maximum of two years. Items are listed in the order defined by your site. Hovering over a result will display the normal range, unit of measure, and the date and time the result was entered.

Notice that each result label is a hyperlink. Clicking a result label will display the graph view for that result. The graph will display all data points for the last two years. In the graph view, you can hover over a data point to see the actual result, unit of measure, and date and time it was entered. There is also a table view of the results on the right side. The results in the table are in chronological order.

Click the Back arrow or Home button to return to the main summary. Clicking the section header will take you to the Flowsheet section of the chart. I can also quickly document new results from the MPage by clicking Add and launching the Ad Hoc Charting window. I can also use the drop down list to select from a site defined list of PowerForms that can be launched by clicking on the name of the form.

I want to enter some information on the Adult Ambulatory Intake form. For a quick demonstration I'm just going to enter temperature and blood pressure. Once I click chart the form closes, I'm back to the summary screen. Now you can see the new results I just charted are displayed.
Component Details

Heading

  • Vitals and Measurements
  • Display "Last 2 years" to the right of the heading
  • Component heading will link to appropriate component in the chart
  • +Add will launch the Ad Hoc charting window
  • Drop down (blue arrow) will display client defined list of PowerForms

Layout

  • First Column - vital sign result name
  • Second Column - most recent result
  • Display a vertical colored line separator between the most recent and previous results
  • Third and Fourth Columns - previous results

Content

  • Display Temp (temperature), BP (blood pressure), HR (heart rate), and other client defined clinical events (maximum of 50)
  • Only display row if a result has been charted
  • Display out-of-range results in appropriate color and with abnormal indicator

Out of Range Results

  • Critical High - bold red font with red exclamation point icon
  • Abnormal High - bold orange font with orange up arrow icon
  • Abnormal Low - bold blue font with blue down arrow icon

Data Format

  • Look Back Range: client defined (default = last two years)
  • Data Available: display three most current results for each item
  • Sort Order: client defined
  • Date / Time Display: display the elapsed time since the result was documented, beneath the result as "x time ago"

Graphing

  • Display the result label as a hyperlink
  • Clicking on the result label will display the graph view
  • Graph will display all data points for the last two years
  • Hover over data points to display the actual result, unit of measure, and date/time
  • Display table of results to the right of the graph
  • Display table column headings as "Date" and "unit of measure appropriate for the result"
  • Display results in chronological order

Hover

  • Display normal range, unit of measure, and date
