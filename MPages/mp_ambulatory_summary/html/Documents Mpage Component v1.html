This page last changed on Jan 15, 2010 by SM015827.

The documents component displays all documents completed on the patient for the previous two years. Each site will determine what note types will be displayed. Notes are sorted in reverse chronological order by the completed date and time. Hovering over a note will display the note type name, subject, status, and completed date and time.

Click the note type name to view the document. I can zoom in and out on the document, and click the back arrow button or home button when I'm ready to return to the summary page. Clicking the Documents section header will take me to the Document Viewing component in the chart. I also have a couple options to start a new note directly from the summary screen. I can click Add and that will take me to Document Viewing with the new note dialog started. I can also select one of my PowerNote favorites from the drop down list to open the note directly.

After documenting a couple items I'll sign the note and close it. The note I just completed is now displayed on the summary screen.
Component Details

Heading

  • Documents (n)
    n = total number of documents displayed
  • +Add button that will open the new note dialog in Document Viewing
  • Drop down list (blue arrow) of the user's PowerNote favorites that can be launched
  • Component heading will link to appropriate component in the chart

Content

  • Note Type (hyperlink to the note)
  • Subject
  • Author
  • Date

Data Format

  • Look Back Range: last two years
  • Note Types to Display: client defined
  • Status: all statuses except InErrored
  • Sort Order: documents listed in reverse chronological order by result date/time
  • Scrolling: client defined
  • Truncate / Wrap: text will wrap if it exceeds the column width

Hover

  • Note Type Name
  • Subject
  • Status
  • Result Date/Time
