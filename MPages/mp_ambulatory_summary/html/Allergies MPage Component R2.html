Overview

The Allergies component displays all active and proposed allergies for the lifetime of the patient. If an allergy has any associated reactions, they are displayed to the right of the allergy name.

Allergies are sorted by severity, and then alphabetically. The highest severity allergies display in a bold, red font.

Hovering over an allergy displays the allergy name, reactions, severity, status, onset date, and any comments. Clicking on the section header takes you to the Allergy component in the chart, and clicking on the Add button takes you to the Allergy component with the Add Allergy conversation already launched.
Component Details

Heading

  • Allergies(n Active)
    n = total number of active allergies
  • Component heading links to appropriate component in the chart
  • +Add button opens the add conversation in the Allergy tab

Content

  • Display allergy name
  • Display associated reactions to the allergy

Data Format

  • Look Back Range: lifetime of the patient
  • Status: display active and proposed allergies
  • Sort Order: sorted by severity, and then alphabetically. Display allergies and associated reaction with anaphylaxis and severe severities in a bold, red font.
  • Scrolling: client-defined
  • Truncate / Wrap: text wraps if it exceeds the column width

Hover

  • Allergy: followed by the name of the allergy with anaphylaxis and severe severities in bold red font.
  • Reactions: followed by the reaction with anaphylaxis and severe severities in bold red font.
  • Severity: followed by the anaphylaxis and severe severities in bold red font.
  • Status: followed by the status of the allergy
  • Onset Date: followed by date of onset
  • Comments: followed by all comments for the allergy (first line: documented date - documented by name / second line: comment text).
