Overview

The Problems component displays all active problems for the lifetime of the patient sorted alphabetically. If there is a codified nomenclature, the code is displayed in parenthesis next to the name.

Positioning the mouse over a problem displays the problem name, onset date, and any comments. Clicking on the section header takes you to the Problems & Diagnoses List in the chart. You can add a new problem from here.
Component Details

Heading

  • Problems (n Active)
    n = total number of active problems
  • Component heading links to appropriate component in the chart

Content

  • Display problem name
  • Display associated code in parenthesis

Data Format

  • Look Back Range: lifetime of the patient
  • Status: display only active problems
  • Sort Order: sort problems in alphabetical order
  • Scrolling: client-defined
  • Truncate / Wrap: text wraps if it exceeds the column width

Hover

  • Problem: followed by the name of the problem.
  • Annotated Display: followed by the annotated display name of the problem.
  • Onset Date: followed by date of onset.
  • Comments: followed by all comments for the problem (first line: documented date - documented by name / second line: comment text).
