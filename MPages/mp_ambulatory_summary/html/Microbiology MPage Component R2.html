Overview

The Microbiology component displays microbiology-related clinical results for the patient. Results are sorted in reverse chronological order. The Within column displays the elapsed time since the test was completed, but you can hover over an item to display the source or body site, actual collected date and time, and susceptibility.

You can click on an item to see the full report. From here you can click the back arrow or home button to return to the summary page. Clicking the section header takes you to either the MicroViewer or the Flowsheet, depending on what your site has implemented.

Component Details

Heading

  • Microbiology (n) Last 6 months
    n = total number of microbiology items
  • Component heading will link to appropriate component in the chart

Content

  • Source / Site
  • Collected within (time elapsed since specimen was collected, link to the report)
  • Normality
  • Status

Data Format

  • Look Back Range: last 6 months (not encounter specific)
  • Status: display all statuses except In Error
  • Sort Order: display in reverse chronological based on collected date/time
  • Truncate / Wrap: text wraps if it exceeds the column width
  • Scrolling: client-defined
  • Date / Time: display time elapsed since specimen was collected

Hover

  • Source/Body Site: followed by source/body site name
  • Collected Date/Time: followed by collected date and time
  • Susceptibility: State if done or not done
