The Problems component displays all active problems for the lifetime of the patient. If there is an associated ICD-9 (or SNO-MED, ICD-10, etc.) code it will display in parenthesis next to the name.
Problems are sorted alphabetically, and if there are more than 10 active problems a scroll bar will appear.

Positioning the mouse over a problem will display the problem name, onset date, and any comments. Clicking on the section header will take you to the Problems & Diagnoses List in the chart.

From here I can add a new problem. To make things quick I'll utilize my favorite folders to add a problem of asthma. Once that is signed, I'll go back to the Ambulatory Summary by clicking in the Table of Contents. I refresh my view, and my new problem is displayed.
Component Details

Heading

  • Problems (n active)
    n = total number of active problems
  • Component heading will link to appropriate component in the chart

Content

  • Display active problem name
  • Display associated code in parenthesis

Data Format

  • Look Back Range: lifetime of the patient
  • Status: display only active problems
  • Sort Order: sort problems in alphabetical order
  • Scrolling: add scroll bar if there are more than 10 results by default, client may change
  • Truncate / Wrap: text will wrap if it exceeds the column width

Hover

  • Problem Name
  • Annotated Display Name
  • Date of Onset
  • Comments
