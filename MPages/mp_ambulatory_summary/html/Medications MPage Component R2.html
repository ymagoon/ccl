Overview

The Medications component displays all active home medications and prescriptions for the lifetime of the patient sorted alphabetically. Medications displays additional order details (such as number of dose refills).

Positioning the mouse over a medication displays the medication name, ordered as mnemonic, order date, signature line, who it was ordered by and what type. Clicking the section header takes you to the Medication List in the chart, and clicking on Add takes you to the Medication List with the Add Order conversation already launched.

From this component, you are able to renew, cancel or complete any of the medications in the list.
Component Details

Heading

  • Medications (n)
    n = total number of active prescriptions or documented medications
  • Component heading will link to appropriate component in the chart
  • +Add will launch the add orders conversation in the Medication List

Content

  • Display label of Rx or Hx: followed by the medication name
  • Display dose, route, frequency, and number of refills

Data Format

  • Look Back Range: lifetime of the patient
  • Status: display only active prescriptions and home medications
  • Truncate / Wrap: text wraps if it exceeds the column width
  • Scrolling: client-defined

Hover

  • Medication: followed by the name of the medication.
  • Ordered As: followed by the display as it was ordered.
  • Order Date: followed by the date of the order.
  • SIG: followed by the signature line.
  • Ordered By: followed by the ordering physician.
  • Order Type: followed by the home medication type (prescribed or documented).
