The Diagnoses component displays all active diagnoses for the current encounter with the associated code in parenthesis next to the name.
Diagnoses are sorted in alphabetical order. (Sort order is stated incorrectly in the video. The video will be updated shortly.)

Positioning the mouse over a diagnosis will display the diagnosis name, date, and any comments. Clicking the section header will take you to the Problems & Diagnoses List in the chart.

From here I can add a new diagnosis, or convert an existing problem to a diagnosis. Once that is signed I'll go back to the Ambulatory Summary by clicking in the Table of Contents. I refresh the view, and my new diagnosis is displayed.
Component Details

Heading

  • Diagnoses (n active)
    n = total number of active diagnoses
  • Component heading will link to appropriate component in the chart

Content

  • Display active diagnosis name
  • Display associated code in parenthesis

Data Format

  • Look Back Range: current, selected encounter
  • Status: display only active diagnoses
  • Sort Order: Display diagnoses in alphabetical order
  • Truncate / Wrap: text will wrap if it exceeds the column width
  • Scrolling: Client Defined

Hover

  • Diagnosis Name
  • Annotated Display
  • Date of Onset
  • Comments
