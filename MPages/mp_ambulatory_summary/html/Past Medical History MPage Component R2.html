Overview

The Past Medical History component displays all resolved problems for the lifetime of the patient sorted alphabetically. If there is a codified nomenclature, the code is displayed in parenthesis next to the name.

Positioning the mouse over a problem displays the problem name, onset date, resolved date, status and any comments. Clicking the section header takes you to the Past Medical History control in the chart.

Component Details

Heading

  • Past Medical History (n)
    n = total number of past medical history items
  • Component heading links to appropriate component in the chart

Content

  • Display problem name
  • Display associated code in parenthesis

Data Format

  • Look Back Range: lifetime
  • Status: display active and resolved problems
  • Sort Order: alphabetical
  • Truncate / Wrap: text will wrap if it exceeds the column width
  • Scrolling: client defined

Hover

  • Problem: followed by the name of the problem.
  • Annotated Display: followed by the annotated display of the problem.
  • Onset Date: followed by the date of onset.
  • Resolved Date: followed by date of resolution for the problem.
  • Status: followed by the status of the problem.
  • Comments: followed by all associated comments for the problem (first line: documented date - documented by name / second line: comment text).
