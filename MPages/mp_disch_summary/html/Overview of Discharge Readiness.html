R1 MPages Help Pages : Overview of Discharge Readiness
This page last changed on Dec 14, 2010 by JP3026.

The discharge process is supported by two MPages views - the Discharge Readiness Dashboard and the Discharge Summary. The Discharge Readiness Dashboard is accessed from the Organizer level of PowerChart®, and it provides a population-based view of key discharge activities, which allows clinicians to get a quick view of their patient's progress toward discharge. The Discharge Summary is accessed from a tab within PowerChart, and it provides a patient-specific view of detailed information regarding the components pertinent to discharge.

At this time, the individual components on this MPages view do not honor privileges and organizational security for an encounter. This situation should be considered when determining the components to display, and which user positions should be given access (see the Preference Maintenance section in the Reference page). From the summary and population views, clinicians can add documentation, add a discharge order, and they can link to "add" modules within the patient chart. The primary objective of the Dashboard and the Summary is to streamline the discharge work flow. In the initial phase of discharge readiness, the Win32 Depart Process needs to be accessed to generate the final discharge documentation. In future phases, the Discharge Readiness Dashboard and the Discharge Summary MPages view will replace the current Win32 Depart Process application.
